如何获取并配置 API 信息

本游戏需要连接一个大型语言模型（LLM）来动态生成剧情。您需要从 AI 服务提供商处获取 API 信息。

步骤 1: 选择一个服务商

您可以选择任意兼容 OpenAI API 格式的服务商，例如：

  • DeepSeek (推荐)
  • OpenAI
  • 以及其他许多第三方或本地部署的模型服务。

步骤 2: 获取 API Key

登录您选择的平台，进入 API 管理页面，创建一个新的 API 密钥（API Key）并复制它。

步骤 3: 填写信息

  • 接口地址: 这是服务商提供的 API 入口点。通常以 `v1` 结尾，例如 `https://api.deepseek.com/v1`。
  • 模型名称: 这是您希望使用的具体模型，例如 `deepseek-chat` 或 `gpt-4-turbo`。
  • API 密钥: 将您复制的密钥粘贴到此处。
